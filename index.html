About App


LibRequest is the app for library users to request books for purchase in their library without filing up long entry forms.

The app uses ISBN as seed to fetch out the full bibliographic details of the book and saves the records. The added records can be downloaded as PDF file.

ISBN can be either entered directly in the inputbar or can be scanned using camera.

App Fetures
  • Scanning ISBN Barcode
  • Input ISBN code
  • Save the records
  • Download added records as a prinatble PDF

What is ISBN?


International Standard Book Number is a unique identifier used for commercial books. It is encoded in an EAN-13 format. Several national and international agencies provide ISBN to publishers.

You can easily find this code in the back of the book.



  1. ISBN Code
  2. Human Readable Text (optional)
  3. Pricing Supplemental Code
